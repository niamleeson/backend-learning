Loops: for and while

Intuition

A for loop asks an iterable for its next item repeatedly until exhausted. A while loop is a repeated if check. The else clause means 'the loop finished naturally.'

Loops let you repeat actions. Python's for loop iterates over any iterable, while while loops continue until a condition becomes false. Both support break, continue, and an optional else clause.

# for loop with enumerate
fruits = ["apple", "banana", "cherry"]
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# while loop with break
count = 0
while True:
    if count >= 5:
        break
    count += 1

# for/else: else runs if loop completes without break
for n in range(2, 10):
    for d in range(2, n):
        if n % d == 0:
            break
    else:
        print(f"{n} is prime")

# zip for parallel iteration
names = ["Alice", "Bob"]
scores = [95, 87]
for name, score in zip(names, scores):
    print(f"{name}: {score}")
  • Use enumerate() instead of manual index tracking
  • Use zip() to iterate over multiple sequences in parallel
  • for/else: the else block runs only if the loop didn't break
  • Avoid modifying a list while iterating over it

Exercises

  1. Matrix Iteration

    Given a 3x3 nested list, use nested for loops to print each element with its row and column index.

    Hint

    Use enumerate() on both the outer and inner loops.

  2. Collatz Sequence

    Write a while loop that takes a positive integer and applies the Collatz rule (if even, divide by 2; if odd, multiply by 3 and add 1) until reaching 1. Count the steps.

    Hint

    Use while n != 1: and track a counter variable.

Common Problems & Solutions

1 Infinite while loop
Cause: The loop condition never becomes False, often because the variable isn't updated inside the loop.
Solution: Ensure the loop variable is modified each iteration, or use `break` with a condition.
2 RuntimeError: list changed size during iteration
Cause: Adding or removing items from a list while iterating over it.
Solution: Iterate over a copy: `for item in my_list[:]` or build a new list.
