Tuples and Immutability

Intuition

Tuples are frozen lists. Their immutability makes them hashable and slightly more memory-efficient, which is why Python uses them for things like function arguments and dict keys.

Tuples are immutable sequences. They are used for fixed collections of items, as dict keys, and for returning multiple values from functions.

point = (3, 4)
x, y = point          # tuple unpacking

# Named tuples for readability
from collections import namedtuple
Color = namedtuple("Color", ["r", "g", "b"])
red = Color(255, 0, 0)
print(red.r)          # 255

# Tuples as dict keys
grid = {}
grid[(0, 0)] = "origin"
grid[(1, 2)] = "point"

# Single-element tuple needs trailing comma
singleton = (42,)     # not (42) which is just int 42
  • Tuples are hashable (if all elements are hashable), making them usable as dict keys and set members.
  • Immutability means no append, insert, or remove — use list conversion if you need mutation.
  • Tuple packing/unpacking enables elegant multi-value returns and swaps: a, b = b, a.
  • namedtuple adds field names, improving readability over plain index access.

Exercises

  1. Tuple-Based Records

    Define a namedtuple called Student with fields name, grade, gpa. Create a list of 3 students and write a function that returns the student with the highest GPA.

    Hint

    Use max() with key=lambda s: s.gpa.

  2. Coordinate Distance

    Write a function distance(p1, p2) that takes two tuples (x, y) and returns the Euclidean distance between them.

    Hint

    Unpack both tuples and use ((x2-x1)**2 + (y2-y1)**2) ** 0.5.

Common Problems & Solutions

1 `(42)` is an int, not a single-element tuple.
Cause: Parentheses without a comma are just grouping. The comma defines the tuple.
Solution: Use `(42,)` for a single-element tuple.
2 Trying to modify a tuple element raises `TypeError: 'tuple' object does not support item assignment`.
Cause: Tuples are immutable by design.
Solution: Convert to a list, modify, and convert back, or create a new tuple with the changed value.
