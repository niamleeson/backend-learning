Redis Data Structures: Strings, Lists, Hashes, Sets, Sorted Sets

Intuition

Redis data structures are like specialized containers in memory. Pick the right container for your pattern: a dict for objects, a list for queues, a sorted set for rankings.

Redis Data Structures

Redis is an in-memory data store supporting rich data structures: strings, hashes, lists, sets, sorted sets, and more.

import redis

r = redis.Redis(host='localhost', port=6379, decode_responses=True)

# Strings
r.set('user:1:name', 'Alice', ex=3600)
r.incr('page:views')

# Hashes
r.hset('user:1', mapping={'name': 'Alice', 'email': 'a@b.com', 'age': '30'})
user = r.hgetall('user:1')

# Lists
r.rpush('queue:tasks', 'task1', 'task2', 'task3')
task = r.lpop('queue:tasks')

# Sets
r.sadd('tags:post:1', 'python', 'redis', 'backend')
common = r.sinter('tags:post:1', 'tags:post:2')

# Sorted Sets
r.zadd('leaderboard', {'alice': 100, 'bob': 85, 'carol': 92})
top3 = r.zrevrange('leaderboard', 0, 2, withscores=True)

Key Takeaways: - Each data structure has O(1) or O(log N) operations optimized for its use case. - Hashes are ideal for object storage. Lists work as queues. Sorted sets enable leaderboards. - All operations are atomic. Use ex or px parameters for TTL-based expiration.

Exercises

  1. Session Store

    Implement a session store using Redis hashes: create session with user data, set TTL to 30 minutes, and retrieve/validate sessions.

    Hint

    r.hset('session:token', mapping=data), r.expire('session:token', 1800), r.hgetall to retrieve.

  2. Real-Time Leaderboard

    Build a leaderboard using sorted sets: add scores, get top 10 players, find a player's rank, and update scores.

    Hint

    zadd for scores, zrevrange(0, 9) for top 10, zrevrank for position, zincrby to update.

Common Problems & Solutions

1 Redis returns bytes instead of strings in Python.
Cause: Default redis-py client returns bytes.
Solution: Pass decode_responses=True when creating the Redis client.
2 Memory usage grows unbounded.
Cause: Keys are created without TTL and never cleaned up.
Solution: Set TTL on all ephemeral keys. Use maxmemory and eviction policies like allkeys-lru.
