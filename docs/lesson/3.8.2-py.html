Aggregation Pipeline

Intuition

The aggregation pipeline is an assembly line for data. Each stage takes documents in, transforms them, and passes them to the next stage.

MongoDB Aggregation Pipeline

The aggregation pipeline processes documents through a sequence of stages, each transforming the data. It is MongoDB's equivalent of SQL GROUP BY, JOIN, and subqueries.

from pymongo import MongoClient

db = MongoClient()['myapp']

pipeline = [
    {'$match': {'status': 'completed', 'date': {'$gte': '2024-01-01'}}},
    {'$group': {
        '_id': '$category',
        'total_revenue': {'$sum': '$amount'},
        'avg_order': {'$avg': '$amount'},
        'count': {'$sum': 1}
    }},
    {'$sort': {'total_revenue': -1}},
    {'$limit': 10}
]
results = list(db.orders.aggregate(pipeline))

# Lookup (JOIN equivalent)
pipeline = [
    {'$lookup': {
        'from': 'users',
        'localField': 'user_id',
        'foreignField': '_id',
        'as': 'user'
    }},
    {'$unwind': '$user'},
    {'$project': {
        'order_total': 1,
        'user_name': '$user.name'
    }}
]

Key Takeaways: - $match filters early to reduce documents flowing through the pipeline. - $group aggregates by a key. $lookup joins collections. $unwind flattens arrays. - $project reshapes output, including/excluding fields and computing new ones.

Exercises

  1. Top Customers Report

    Write a pipeline that finds the top 5 customers by total spending, including their name from a users collection via $lookup.

    Hint

    $group by user_id with $sum, $sort descending, $limit 5, then $lookup to users for the name.

  2. Time-Series Aggregation

    Aggregate events by day using $dateToString in $group, computing counts and averages per day.

    Hint

    $group: {'_id': {'$dateToString': {'format': '%Y-%m-%d', 'date': '$created_at'}}}.

Common Problems & Solutions

1 Aggregation is slow on large collections.
Cause: No index on fields used in $match, or $match is not the first stage.
Solution: Place $match as early as possible so it can use indexes. Create indexes on filtered fields.
2 $lookup returns an empty array instead of matching documents.
Cause: Field types do not match: one side stores strings, the other ObjectIds.
Solution: Ensure localField and foreignField have the same BSON type, or use $toObjectId in a pipeline $lookup.
